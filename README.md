

# Sales Performance Dashboard

## Project Overview
Developed an interactive Sales Performance Dashboard using Power BI to analyze key business metrics such as sales, profit, quantity, and profit margin. The dashboard helps in understanding business performance and supports data-driven decision-making.


##  Objective
To analyze sales performance across categories, regions, and products, and identify key trends and insights for business improvement.


##  Tools & Technologies
- Power BI  
- SQL  
- Excel  
- Power Query  
- DAX  



##  Key Features
- KPI cards for Total Sales, Profit, Quantity, and Profit Margin  
- Sales trend analysis over time (Line Chart)  
- Category-wise performance analysis (Bar Chart)  
- Region-wise sales distribution (Pie Chart)  
- Top 5 products by sales  
- Scatter chart for Sales vs Profit analysis  
- Interactive slicers for filtering data  



##  Key Insights
- Sales and profit are increasing overall  
- Profit margin shows a slight decline  
- Electronics is the top-performing category  
- Regional sales are evenly distributed  
- Some products have high sales but low profit
